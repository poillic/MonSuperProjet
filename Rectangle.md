# Rectangle

Créez une classe `Rectangle` avec les propriétés suivantes :
- `Width` : de type `float`, pour stocker la largeur du rectangle.
- `Height` : de type `float`, pour stocker la hauteur du rectangle.

Ajoutez un constructeur qui prend en paramètres la largeur et la hauteur du rectangle et initialise les propriétés correspondantes.

    // Calculer et retourner le périmètre du rectangle
